# ChartMania - Frontend

Welcome to **ChartMania**.

ChartMania is an application designed to help users generate beautiful, useful charts quickly and easily. This repository contains the **frontend** codebase.

## 🚧 Project Status
**Under Development**

## 🛠 What’s inside?
- **Registration**: Guests can create an account.
- **JWT Authentication**: Secure login using JSON Web Tokens (JWT).
- **Refresh Token Handling**: Automatic token refresh to keep sessions active securely.
- **Personal Area**: A dedicated user space.
- **User Charts Table**: User can see their charts.
- **Chart Deletion**: Users can **remove** charts from their personal collection.
- **Chart Creation**: Users can **create** new charts.
- **Chart Editing**: Users can **edit** existing charts.
- **Chart Image**: Users can **export** charts as images.
